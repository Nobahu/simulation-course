### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

### Реализация

В данной лабораторной работе мною был использовал **линейный конгруэнтный метод** для генерации случайных величин в диапазоне [0,1]. Также для сравнения был взял **mt19937_64** из библиотеки <random> C++. Объем выборки, согласно заданию — **100 000 значений**, однако я запускал генераторы на следующих значениях, в рамках тестов [1000,10000,100000,1000000].
