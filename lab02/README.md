**Задание:**

Предусмотреть возможность ввода шага моделирования и вывода результатов.

**Вывод**



**По результатам проведенного моделирования полета тела в атмосфере при различных значениях dt можно сделать следующие выводы:**



* **Уменьшение шага моделирования дает уменьшение погрешности при расчетах**
* **При переходе от dt = 0.01 до dt = 0.001 разница между получившимися значениями - минимально. Из этого следует, что дальнейшее уменьшение шага моделирование нецелесообразно, т.к полученный прирост точности несоответствует потраченному времени. Оптимальный шаг для моделирования — 0.01-0.001**
