## Моделирование полёта тела в атмосфере

### Задание

Реализовать приложение для моделирования полёта тела в атмосфере.  
Предусмотреть возможность ввода шага моделирования и вывода результатов.

Выполнить моделирование **без очистки предыдущих результатов** для различных шагов моделирования, сравнить траектории и заполнить таблицу:

### Отчет

|Шаг моделирования, с|1|0.1|0.01|0.001|0.0001|
|-|-|-|-|-|-|
|Дальность полёта, м|34,93|44,57|45,51|45,60|45,60|
|Максимальная высота, м|5,64|14,41|15,43|15,53|15,54|
|Скорость в конечной точке, м/с|15,53|19,02|19,14|19,15|19,16|

### Вывод

По результатам проведенного моделирования полета тела в атмосфере при различных значениях dt можно сделать следующие выводы:



* Уменьшение шага моделирования дает уменьшение погрешности при расчетах
* При переходе от dt = 0.01 до dt = 0.001 разница между получившимися значениями - минимально. Из этого следует, что дальнейшее уменьшение шага моделирование нецелесообразно, т.к полученный прирост точности несоответствует потраченному времени. Оптимальный шаг для моделирования — 0.001
